Flexbox 是现代 CSS 中最强大的布局工具之一。以下示例展示如何使用 Flexbox 实现水平和垂直居中。

<CodeDemo
  code={`<div class="flexbox-demo-inline">
  <div class="demo-item">1</div>
  <div class="demo-item">2</div>
  <div class="demo-item">3</div>
</div>`}
  lang="html"
>
  <div class="flexbox-demo-inline">
    <div class="demo-item">1</div>
    <div class="demo-item">2</div>
    <div class="demo-item">3</div>
  </div>
</CodeDemo>

## 关键属性说明

| 属性              | 作用              | 值示例                    |
| ----------------- | ----------------- | ------------------------- |
| `display: flex`   | 开启 Flexbox 布局 | —                         |
| `justify-content` | 主轴对齐方式      | `center`, `space-between` |
| `align-items`     | 交叉轴对齐方式    | `center`, `stretch`       |
| `gap`             | 子项间距          | `12px`, `1rem`            |
